# **My Coachee**

![multi platform](add-later-on)

## **Project Goal**
This is a platform to help coaches to keep their sessions organized. A coach can add all their coachees and then book sessions with them. In a session, there is everything they talk about in the previous sessions. So that coaches have a powerful tool in their hands.
